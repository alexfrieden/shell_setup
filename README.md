# shell_setup

## Change Default Shell

```
chsh -s /bin/zsh
```

## Install zsh
URL to install: https://ohmyz.sh/

But most likely it is
```
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
## Install Plug Ins

```vim ~/.vimrc```

Then go in and run `:PlugInstall`
